Getting Started

Set up Pulsar v2.0 in minutes

1

Clone Repository

git clone https://github.com/lmanchu/pulsar.git
cd pulsar && pnpm install
2

Install Native Host

cd apps/native-host
pnpm install && pnpm build
pnpm run postinstall
3

Load Extension

Open Chrome ‚Üí chrome://extensions/ ‚Üí Load unpacked ‚Üí Select extensions/pulsar-extension-v2/

4

Start Development

# Terminal 1: Backend
cd apps/web && pnpm dev

# Terminal 2: Native Host
cd apps/native-host && pnpm dev
